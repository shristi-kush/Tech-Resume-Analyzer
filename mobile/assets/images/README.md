# Brand assets

| File | Use |
|------|-----|
| `logo.png` | App logo, icon, splash, in-app branding |
| `banner.png` | Legacy banner (optional) |
| `profile.png` | Default avatar placeholder |
| `empty-state.png` | Empty state illustration |

Replace `logo.png` (512×512 or 1024×1024 PNG) to update branding everywhere.
